## 复杂度评估字段说明

proposal.md 的 YAML frontmatter 中包含复杂度评估字段，用于动态调整 spec 树深度：

```yaml
complexity: low | medium | high  # 复杂度级别
estimated_depth: 2 | 3 | 4       # 预估的 spec 树深度

# 复杂度评估维度（自动计算）:
complexity_factors:
  task_count: 0-10               # 任务数量
  code_change_lines: 0-500       # 代码变更行数
  module_count: 0-5              # 涉及模块数
  dependency_change: 0-3         # 依赖变更数
  security_impact: none | indirect | direct  # 安全影响
```
