# sop-code-refactor 触发测试

## 应该触发

### 英文触发词
- "refactor code"
- "refactor"
- "clean code"
- "improve code structure"
- "optimize code"

### 中文触发词
- "重构"
- "重构代码"
- "代码优化"
- "改进代码结构"
- "清理代码"

### 改写/同义词
- "这段代码太乱了，帮我整理一下"
- "优化这个方法的实现"
- "提取公共逻辑"

## 不应触发

### 无关查询
- "写新功能"
- "分析需求"
- "查找bug"

### 其他 Skill 的触发词
- "分析bug" (应触发 sop-bug-analysis)
- "升级依赖" (应触发 sop-dependency-manager)
- "写测试" (应触发 sop-test-implementation)

## 边界情况

### 可能误触发
- "优化" (可能是性能优化，应询问用户意图)
- "改进" (可能是功能改进)

### 可能漏触发
- "这段代码有重复" (隐含重构)
- "这个方法太长了" (隐含提取方法)
