# sop-progress-supervisor 功能测试

## 测试 1: 进度查询

### 输入
```
查询当前工作流进度
```

### 预期输出
- 显示当前阶段
- 显示已完成阶段
- 显示待处理阶段
- 显示阻塞问题（如有）

### 验证点
- [ ] 是否显示了当前状态
- [ ] 是否显示了阶段进度
- [ ] 是否识别了阻塞问题
