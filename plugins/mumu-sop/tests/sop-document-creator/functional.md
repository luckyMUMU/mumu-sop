## 测试 2: API 文档创建

### 输入
```
文档类型：api
API 名称：用户管理 API
端点：
- GET /users - 获取用户列表
- POST /users - 创建用户
- GET /users/{id} - 获取用户详情
- PUT /users/{id} - 更新用户
- DELETE /users/{id} - 删除用户
```

### 预期输出
- 生成 API 文档
- 包含所有端点描述
- 包含请求/响应示例
- 包含错误码说明

### 验证点
- [ ] 是否包含所有端点
- [ ] 是否包含请求参数说明
- [ ] 是否包含响应示例
- [ ] 是否包含错误码
