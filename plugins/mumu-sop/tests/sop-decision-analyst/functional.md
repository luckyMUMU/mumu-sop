# sop-dual-cycle-decision 功能测试

## 测试 1: 复杂需求决策流程

### 输入
```
需求：构建一个企业级 CRM 系统
- 客户管理
- 销售管道
- 报表分析
- 第三方集成（邮件、短信）
预算有限，需在 3 个月内上线 MVP
```

### 预期行为
1. 执行外循环：多轮提问分析意图
2. 记录决策路径
3. 创建执行计划
4. 执行内循环：审查和预演
5. 生成审查意见
6. 等待用户确认

### 验证点
- [ ] 是否进行了 4 轮提问
- [ ] 是否记录了决策路径
- [ ] 是否创建了执行计划
- [ ] 是否进行了内循环审查
- [ ] 是否等待用户确认
