# fast-responder 触发测试

## 应该触发

### 英文触发词
- "what is the capital of France"
- "convert JSON to YAML"
- "quick question"
- "simple lookup"
- "status check"
- "format this text"
- "quick help"

### 中文触发词
- "快速问题"
- "格式转换"
- "简单查询"
- "状态检查"
- "查看一下"
- "快速帮助"

### 改写/同义词
- "帮我快速查一下"
- "简单问一下"
- "能否快速转换"
- "快速查看状态"

## 不应触发

### 复杂任务
- "帮我实现一个用户系统"
- "设计一个微服务架构"
- "重构这个代码库"
- "分析这个bug的根本原因"

### 需要多步骤的任务
- "帮我写一个完整的API"
- "生成100个测试文件"
- "优化数据库查询性能"

### 其他 Agent 的触发词
- "需要架构决策" (应触发 architect-decider)
- "实现这个功能" (应触发 code-implementer-router)
- "批量处理这些文件" (应触发 batch-processor)

## 边界情况

### 可能误触发
- "快速实现" (可能混淆，需根据上下文判断)
- "简单设计" (可能混淆，需根据上下文判断)

### 可能漏触发
- "查一下北京天气" (需要识别为简单查询)
- "把这个转成CSV" (需要识别为格式转换)
